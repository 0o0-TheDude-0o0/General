# Template Tools Plugin for Unraid 7.2.3+

Convert Docker run commands and docker-compose.yml files directly into Unraid templates.

## Features
- Convert docker run commands to Unraid XML templates
- Parse docker-compose.yml files
- Automatic path mapping to Unraid appdata structure
- Timezone auto-detection
- One-click template creation & deployment

## Installation
1. Place `templateTools.plg` in `/boot/config/plugins/`
2. Install via Unraid Plugin Manager
3. Navigate to Docker tab and click "Template Tools" button

## Usage
1. Go to Docker tab in Unraid
2. Click "Template Tools" button
3. Paste docker command or compose file
4. Click "Convert to Template"
5. Review and deploy
